Binary Search Game

In this project, we used pair programming to make a text based game in JS based on Binary Search. We played a game in class to understand the concept and drew out the algorithm for our game with a flow chart. My group chose to make a game where the program would guess the US state that the user thought of. The program would ask the user questions about geography and position of the state relative to other states or geographic features of the US.

Binary search is the concept of dividing the amount of items in a list by two until there is only one possibility for what the item could be. What that means to this game and the user interface is that the program asks the user yes or no questions until there is only one possibility for what it could be. For my group, we asked as many questions as we could to divide the possibilities by two, and we would use readyesorno functions to ask a yes or no question. The user would write yes or no depending on how the question related to their state. But there were a couple of times when we needed to divide by 3 or 4. For those cases we would use a readchoice function to choose between different terms assigned to a number. The user would write the number that stood for the term that was correct to the state they chose.
